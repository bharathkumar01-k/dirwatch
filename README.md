# Dirwatch

## Prerequisites

1. Typescript
2. NodeJS (min version - 20)
3. Redis
4. MongoDB

## Steps to run

1. Clone the repository
2. Run ***tsc -b***
3. Run ***npm install***
4. Navigate to __/src__ folder
5. Run ***node app.js***
